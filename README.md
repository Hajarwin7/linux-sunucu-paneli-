# Linux Sunucu Durum Paneli

Bu proje, Linux sunucusunun anlık durumunu web üzerinden görüntüleyen bir izleme sistemidir.

## Gösterilen Bilgiler:
- Rapor Saati
- CPU Kullanımı
- RAM Kullanımı
- Disk Kullanımı
- Aktif Kullanıcı Sayısı

## Kullanılan Teknolojiler:
- Bash Script
- Nginx Web Server
- HTML
- Cron Job
